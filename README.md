## **기본 정보**
- **Node.js 버전:** `v20.18.3`
- **Vue.js 버전:** `^3.5.13`
- **패키지 관리자:** `npm`
- **번들러:** `Vite 6.1.0`

---

## **프로젝트 실행**
### **설치**
```sh
npm install
```
### **개발 서버 실행**
```sh
npm run dev
```
### **빌드**
```sh
npm run build
```
### **빌드된 프로젝트 미리보기**
```sh
npm run preview
```
